Stu Bulk Move to Songs (Temporary Admin Tool)
-------------------------------------------------

Adds Tools → Bulk Move to Songs so you can:
- Dry Run: preview which Posts match your filters
- Convert: change their type to 'song'
- Revert: move any converted items back to 'post'

Filters:
- Match content containing [chordpress (default ON)
- Optional category and tag slug filters
- Batch size control

Install:
1) Upload 'stu-bulk-move-to-songs.php' to '/wp-content/mu-plugins/' (create the folder if missing).
2) In WP Admin, go to Tools → Bulk Move to Songs.
3) Dry run, then Convert.
4) Use Revert if needed.
5) Delete the file when finished.
